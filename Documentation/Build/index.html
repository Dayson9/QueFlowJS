Installation

To install QueFlow, you either can use CDN, go to its GitHub repository or direct download.


Using CDN

CDN is the fastest way to install a library. Just copy and paste the below code into the head section of your HTML file.


<script src="https://cdn.jsdelivr.net/gh/Dayson9/QueFlowJS@main/QueFlow.min.js"></script>

Quick Start

QueFlowJS is a lightweight library which has 3 core methods: iRender(), Render(), and Signal().


Create a reactive data with the Signal() method. The Signal() method only accepts one argument, which is the reactive data.





Easily render UI using the Render() method. This method accepts only two arguments, which is the HTML string to render and the element to render it on.


The API Reference

QueFlow has just 3 core methods for managing and rendering reactive state UI.

  1. Render()
  2. iRender()
  3. Signal()

The Render() method


The Render() method is primarily used for parsing html/jsx content into an element.


Syntax: Render( jsx , app )

The Render() method accepts two arguments as shown above; the JSX/HTML to render and the element the JSX/HTML would be rendered on.

It also supports data-binding. The jsx to render can also contain some binded data.
